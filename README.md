# 🌤️ Weather App

A simple and responsive **Weather Application** built using **React + Vite** that allows users to search for any city and view the current weather conditions instantly.

The application fetches real-time weather data using the **OpenWeather API** and displays important weather information in a clean UI.

---

## 🚀 Live Demo

🔗 **Deployed Website:**
https://feel-the-skyy.netlify.app/

---

## 📌 Features

* 🌍 Search weather by city name
* 🌡️ Displays temperature in real time
* ☁️ Shows weather condition (Cloudy, Clear, Rain, etc.)
* 💧 Humidity information
* 💨 Wind speed display
* ⚡ Fast performance using Vite
* 📱 Responsive design

## ⚙️ Installation & Setup

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/weather-app.git
```

### 2️⃣ Navigate to the project folder

```
cd weather-app
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Add environment variable

Create a `.env` file in the root folder and add:

```
VITE_API_KEY=your_openweather_api_key
```

### 5️⃣ Run the development server

```
npm run dev
```

The app will run at:

```
http://localhost:5173
```
